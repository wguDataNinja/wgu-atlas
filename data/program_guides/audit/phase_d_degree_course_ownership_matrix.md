# Phase D Degree/Course Ownership Matrix

**Date:** 2026-03-21

| Field | Ownership | Decision Rationale |
|---|---|---|
| Standard Path course order | Degree only | Core degree sequencing value; not independently trustworthy on course pages without program context. |
| Standard Path CUs | Degree only | Degree plan context; course pages already have catalog CU concepts and may differ by program context. |
| Standard Path term values | Degree only | Term meaning is program-sequence specific. |
| AoS groups | Degree only | Program-structure context, not course identity. |
| Guide course descriptions | Degree now, Course later | High-value narrative; course-page use blocked until Phase E matching. |
| Competency bullets | Degree now, Course later | Useful outcomes detail; dense, should be collapsible. Course-page use blocked until Phase E. |
| Capstone identification/details | Degree only | Program-completion context, not standalone course canonicality. |
| Guide version/pub metadata | Degree internal-facing optional | Trust/provenance marker; low direct student value. |
| Prerequisite mentions (parsed) | Internal only | Known false-positive risk. |
| Certification-prep mentions (structured) | Internal only for v1 | No stable structured field yet; defer extraction pass. |
| Course role within degree (derived) | Degree only | Strong student value for degree navigation. |
| Cross-program appearance context from guides | Internal only in Phase D | Requires normalized course matching; Phase E+ concern. |
| Family-specific specialty sections | Internal only in Phase D | Too variant-heavy for conservative first publish. |
